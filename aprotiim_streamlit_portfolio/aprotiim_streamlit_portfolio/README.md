# Aprotiim Joardar Streamlit Portfolio

A polished multi-page Streamlit portfolio site with recruiter-focused storytelling.

## Pages
- About
- Projects
- Work Experience
- Education
- Contact

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Notes
- Replace the LinkedIn placeholder with your real profile URL.
- You can deploy this on Streamlit Community Cloud, Render, or any VM/container.
